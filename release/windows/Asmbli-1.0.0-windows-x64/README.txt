Asmbli v1.0.0 Beta - Windows Release
======================================

Thank you for downloading Asmbli! This is an early access beta build.

INSTALLATION:
1. Extract all files to a folder of your choice
2. Run "Asmbli.exe" to start the application
3. Follow the setup wizard to configure your AI providers

SYSTEM REQUIREMENTS:
- Windows 10 64-bit or Windows 11
- 8GB RAM minimum (16GB recommended)
- 10GB available disk space (for AI models)

WHAT'S INCLUDED:
- Multi-model AI chat (Claude, OpenAI, Ollama)
- Basic agent templates (experimental)
- Document context upload
- Local data storage
- Secure API key storage

GETTING STARTED:
1. Launch Asmbli.exe
2. Add your API keys for:
   - Anthropic Claude (recommended)
   - OpenAI GPT models
   - Or configure local Ollama models
3. Create your first agent or start chatting!

BETA LIMITATIONS:
- This is early access software - expect bugs
- Agent responses may hallucinate
- MCP integration is experimental
- Document context may get lost in long conversations
- Limited error handling

SUPPORT:
- Report bugs: https://github.com/WereNext/Asmbli/issues
- Documentation: https://github.com/WereNext/Asmbli
- Community: https://github.com/WereNext/Asmbli/discussions

PRIVACY:
- No data collection or telemetry
- All conversations stored locally on your device
- API keys secured in Windows Credential Manager
- Direct connection to AI providers (no proxy)
