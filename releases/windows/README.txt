Asmbli - AI Agent Builder for Windows
======================================

Version: 1.0.0
Build Date: 2025-09-18
Platform: Windows x64

INSTALLATION INSTRUCTIONS:
=========================

1. Extract this entire folder to your desired location
   (e.g., C:\Program Files\Asmbli\)

2. Run agentengine_desktop.exe to start the application

3. The application will create data folders in:
   %APPDATA%\AgentEngine Inc\AgentEngine Desktop\

SYSTEM REQUIREMENTS:
===================

- Windows 10 or later (x64)
- Minimum 4GB RAM (8GB recommended)
- 500MB free disk space
- Internet connection for AI model access

FEATURES:
=========

✅ Professional AI Agent Builder
✅ 60+ MCP Server Integrations (GitHub, Microsoft 365, AWS, etc.)
✅ Multi-color scheme design system
✅ Real-time chat interface with streaming
✅ Enterprise security with OAuth 2.0
✅ Vector knowledge base integration
✅ Cross-platform agent templates

FIRST RUN:
==========

1. Complete the onboarding process
2. Configure your first AI model (Claude, GPT, or local models)
3. Add integrations to connect your favorite services
4. Create your first agent using templates or build from scratch
5. Start chatting to test your agent
