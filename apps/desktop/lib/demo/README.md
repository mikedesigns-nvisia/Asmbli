# Demo Branch: Confidence Microscopy & Reasoning Intelligence

This branch contains demo implementations showcasing Asmbli's unique confidence-aware reasoning capabilities.

## Demo Scenarios

### 1. VC/Investor Demo (8 minutes)
- **File**: `scenarios/vc_demo.dart`
- **Focus**: "Holy Shit" moment - AI reasoning transparency
- **Key Features**: Real-time confidence, uncertainty intervention

### 2. Enterprise Demo (12 minutes)
- **File**: `scenarios/enterprise_demo.dart`
- **Focus**: ROI and cost savings
- **Key Features**: Cost-aware routing, compliance workflows

### 3. Technical Demo (15 minutes)
- **File**: `scenarios/technical_demo.dart`
- **Focus**: Architecture and implementation
- **Key Features**: Multi-model consensus, local/cloud routing

## Demo Data

### Sample Documents
- `data/sample_contracts/` - Legal documents with compliance complexity

- `workflows/legal_analysis.json` - Contract analysis with confidence hotspots
- `workflows/investment_memo.json` - Investment analysis with uncertainty triggers

## Demo Components

### Confidence Visualization
- `components/confidence_microscopy_widget.dart` - Hierarchical confidence display
- `components/uncertainty_intervention_panel.dart` - Human consultation UI
- `components/multi_model_consensus_view.dart` - Model agreement visualization

### Demo Mode Toggle
- `services/demo_mode_service.dart` - Switch between demo and production
- `data/demo_state_manager.dart` - Manage demo scenarios and progression

## Usage

```bash
# Switch to demo branch
git checkout demo/confidence-microscopy

# Run in demo mode
flutter run --dart-define=DEMO_MODE=true

# Specific demo scenario
flutter run --dart-define=DEMO_SCENARIO=vc_demo
```
